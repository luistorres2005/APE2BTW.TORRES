Análisis Comparativo

HTML5 vs HTML 4.01

En esta comparativa verás qué etiquetas HTML utilizábamos antes y cuáles las reemplazan hoy en día (semántica o funcionalmente).

Tabla de Etiquetas

Etiqueta Obsoleta Uso Antiguo Etiqueta Moderna Uso Actual
<font> Estilo de texto (color, tamaño, tipo). <span> + CSS Aplicar estilos mediante clases CSS.
<center> Centrar contenido. CSS Propiedades como text-align: center o Flexbox.
<b> Negrita visual. <strong> Marcar contenido importante (semántico).
<strike> Texto tachado. <del> Marcar texto eliminado o corregido.
<big> Texto grande. CSS Propiedad font-size.
<i> Texto en cursiva. <em> o CSS Enfásis en el texto o estilos visuales.

Conclusión

A medida que la web maduró, muchas etiquetas dejaron de ser recomendadas porque mezclaban contenido y presentación. Comparar etiquetas obsoletas con sus alternativas modernas ayuda a escribir código más limpio, semántico y preparado para el futuro.
