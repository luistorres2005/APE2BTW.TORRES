Hitos en la historia del HTML

El desarrollo del HTML no fue lineal; estuvo marcado por guerras de navegadores y cambios de filosofía.

Desde los primeros documentos estáticos hasta las aplicaciones web que usamos a diario, HTML ha ido evolucionando para responder a nuevas necesidades. Conocer esta línea de tiempo permite entender por qué hoy trabajamos con HTML5.

HTML 1.0 - 1993

  • Diseñado por Tim Berners-Lee.
  • Era puramente estructural y muy limitado.
  • Su objetivo era conectar documentos científicos mediante hipervínculos.
  • No existían las imágenes integradas ni las tablas.
  • Contaba con 18 etiquetas básicas aproximadamente.

HTML 2.0 - 1995

  • Se formaliza el lenguaje bajo el IETF (Primer estándar oficial).
  • Su gran aporte fue la introducción de los formularios.
  • Permitió por primera vez la interacción usuario-servidor.

HTML 3.2 - 1997

  • Se dio en plena expansión comercial de internet.
  • Se introdujeron las tablas y atributos visuales.
  • Permitió diseños más ricos, pero mezclaba estructura con presentación.

HTML 4.01 - 1999

  • La W3C recomendó separar la estructura del diseño.
  • Se introdujeron versiones Strict y Transitional.
  • Ayudó a los desarrolladores a limpiar su código.

XHTML - 2000

  • Un intento de reformular HTML como XML (sintaxis estricta).
  • Minúsculas obligatorias y etiquetas bien cerradas.
  • Muy educativo, pero resultó demasiado estricto para la web común.

HTML5 - 2014 (Actualidad)

  • Desarrollado por el WHATWG y la W3C.
  • Introduce etiquetas semánticas modernas.
  • Soporte nativo para audio y video sin plugins.
  • Incluye APIs para aplicaciones web completas.
